# Event Scheduling & Resource Allocation System

This Flask-based web application allows organizations to schedule events
and allocate shared resources such as rooms, instructors, and equipment.

## Features
- Add, edit, and view events
- Add, edit, and view resources
- Allocate resources to events
- Conflict detection for overlapping time slots
- Resource utilization report

## Tech Stack
- Python
- Flask
- Flask-SQLAlchemy
- SQLite
- HTML, CSS, Bootstrap

## How to Run
1. Clone the repository
2. Install dependencies:
   pip install -r requirements.txt
3. Run the application:
   python app.py
4. Open browser:
   http://127.0.0.1:5000
